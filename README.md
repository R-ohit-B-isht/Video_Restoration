
# Plate of Rice

A Theme Store for all linux distros


## Run Locally

Clone the project

```bash
  git clone https://github.com/R-ohit-B-isht/PlateOfRice.git
```

Go to the project directory

```bash
  cd PlateOfRice
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run electron-dev
```
